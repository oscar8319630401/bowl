# 🏺 도자기 타임머신

**"교과서 속 납작한 도자기를 살려내자!"**

국사책 사진 한 장으로 외우던 한국 도자기를, 손으로 돌려보고 확대해서 배우는 3D 학습 웹앱입니다.

👉 **[https://oscar8319630401.github.io/bowl/](https://oscar8319630401.github.io/bowl/)**

## 뭘 할 수 있나

- **스와이프 타임라인** — 선사 → 삼국 → 통일신라 → 고려 → 조선을 좌우로 넘기며 이동
- **3D 뷰어** — 도자기를 360도 돌리고 확대. 표면의 반짝이는 점(핫스팟)을 누르면 상감·음각·청화 같은 기법 설명이 뜸
- **플래시카드** — 틴더처럼 스와이프하며 빠르게 암기
- **퀴즈 & 오답노트** — 틀린 문제는 오답노트로 직행
- **마이 갤러리** — 학습·퀴즈로 배지를 모아 빈 진열장을 채움

모바일과 PC를 모두 지원합니다. 진행도는 브라우저에 저장됩니다.

## 도자기 12점

선사(빗살무늬토기·민무늬토기), 삼국(신라 장경호·가야 굽다리접시), 통일신라(인화문 뼈단지),
고려(청자 상감운학문 매병·참외모양 병·음각연화당초문 매병),
조선(분청사기 인화문 항아리·백자 달항아리·청화매죽문 항아리·철화포도문 항아리)

## 3D는 어떻게 만들었나

두 가지를 섞어 씁니다.

1. **실물 스캔 모델** — 국립중앙박물관이 공개한 3D 스캔 데이터를 웹용으로 변환(OBJ → GLB, 약 20MB → 200KB)
2. **프로시저럴 모델** — 스캔 데이터가 없는 유물은 회전체(LatheGeometry)로 형태를 만들고, 무늬는 캔버스에 절차적으로 그려 텍스처로 입힘

도자기는 물레로 돌려 만든 회전체라서, 실루엣 제어점을 곡선 보간해 돌리면 형태가 자연스럽게 나옵니다.

## 개발

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 프로덕션 빌드
npm run typecheck
```

`main` 브랜치에 push하면 GitHub Actions가 자동으로 빌드해 Pages에 배포합니다.
